# Keyper Wallet

Smart contract wallet for iPhone. Hardware wallet security, without the hardware.

## What is Keyper?

Keyper is an iOS-native smart contract wallet that uses iPhone's Secure Enclave for key management. Built on Safe smart contracts, it provides hardware-grade security while supporting 500+ EVM networks with a single address.

## Features

- **Secure Enclave** — Private keys protected by iPhone's security chip
- **Safe Protocol** — Built on battle-tested Safe contracts (v1.4.1, $100B+ secured)
- **500+ Networks** — Same address across all EVM chains via CREATE2
- **Pure Swift** — Zero third-party dependencies, no supply chain risk
- **100% Open Source** — Every line of code is auditable
