Our Principles

What we believe

1

Hardware-Grade Security, No Hardware

vs Ledger

iPhone Secure Enclave provides chip-level protection. Your private key never leaves the security chip. Same security as Ledger, without carrying extra devices.

Carry Ledger everywhere → iPhone is enough
2

100% Open Source

vs Ledger

Every line of code is public and auditable. No hidden logic, no backdoors, no trust required. Verify, don't trust.

Ledger closed-source firmware → Fully auditable
3

Zero Dependencies

vs MetaMask

Only Apple native APIs. No third-party libraries. Zero supply chain risk. No npm packages that could be compromised.

MetaMask: 1000+ npm packages → Pure Swift only
4

User Sovereignty

vs Ledger Recover

No custody, no cloud sync, no upload. Your escape route is your own EOA—not dependent on any service. We never touch your keys.

Ledger Recover controversy → 100% self-custody
5

Minimal Architecture

vs All Wallets

Only two layers: Secure Enclave for daily use, EOA for escape. No feature bloat, no complexity. More code = more bugs.

Complex multi-layer systems → Two layers only
6

No Compromise

Our Choice

iOS only. Swift native only. We don't chase market share at the cost of security. No Android, no React Native, no cross-platform shortcuts.

Cross-platform compromises → Purpose-built for iPhone
7

Battle-Tested Contracts

Safe 1.4.1

Built on Safe smart contracts—the most trusted wallet infrastructure, securing $100B+ in assets since 2018. We use the latest audited version (1.4.1).

Unproven custom contracts → $100B+ battle-tested
